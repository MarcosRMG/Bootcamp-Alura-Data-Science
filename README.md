## Módulo 1: Analisando dados da saúde com Python e Pandas
Seleção dos dados no DATASUS (https://datasus.saude.gov.br/) utilizando o tabnet (https://datasus.saude.gov.br/informacoes-de-saude-tabnet/) referente aos custos de assistência a saúde para tratamento e visualização dos dados.
